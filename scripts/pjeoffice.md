 # PJe Office
```bash
wget -O PJe.zip http://my.opendesktop.org/s/2a65eG4rqf3oLSP/download
unzip PJe.zip
aptinst -y ./*.deb
sudo cp pje-office.desktop /usr/share/applications/pje-office.desktop
rm -rfv *.deb *.desktop *.zip
```
